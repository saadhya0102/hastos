Compute the **parity** of a 32-bit value: `1` if it has an odd number of set bits, else `0`. Parity
bits are a simple error-detection mechanism.

## Your task

```c
int parity(uint32_t x);
```

## Hints (no spoilers)

- Clear the lowest set bit repeatedly (`x &= x - 1`), toggling a flag each time.
- Or XOR-fold: `x ^= x >> 16; x ^= x >> 8; x ^= x >> 4; x ^= x >> 2; x ^= x >> 1; return x & 1;`

`parity(0b1011)` = `1`; `parity(0xFFFFFFFF)` = `0` (32 is even). Use **Run** then **Submit**.
